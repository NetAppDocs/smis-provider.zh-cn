= SMIS 命令
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
`smis crsp` 命令介绍 NetApp SMI-S Provider 支持的已注册 CIM 子配置文件，包括 NetApp SMI-S Provider 子配置文件。

SMIS 命令

` [-t ｛ * http* | * https * ｝ ]`

`C ： \Program Files （ x86 ） \NetApp\SMIs\pegasus\bin`

具有有效用户名和密码的用户

.Parameters
* ` * （ -t ｛ http ： https ｝ ） *`
+
要使用的协议： HTTPS （默认）或 HTTP
