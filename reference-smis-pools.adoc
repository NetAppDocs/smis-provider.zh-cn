= SMIS 池
:icons: font
:imagesdir: ./media/


[role="lead"]
`smis pools` 命令可列出存储系统的存储池。

`sMis Pools`

` [-t ｛ * http* | * https * ｝ ]`

`C ： \Program Files （ x86 ） \NetApp\SMIs\pegasus\bin`

具有有效用户名和密码的用户

.Parameters
* ` * （ -t ｛ http ： https ｝ ） *`
+
要使用的协议： HTTPS （默认）或 HTTP



`smis pools` 命令及其缩写输出：

----

ONTAP_DiskPrimordialPool.InstanceID="ONTAP:0084259609:Other"
