= SMIS Cimserver
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
`smis cimserver` 命令用于启动，停止，重新启动或获取 CIM 服务器的状态。

`smis cimserver`

` ｛ * 开始 * ｝ * 停止 *` * 重新启动 * * * * 状态 * ｝

`C ： \Program Files （ x86 ） \NetApp\SMIs\pegasus\bin`

管理员（ Windows ）

.Parameters
* * 开始 *
+
启动 CIM 服务器。

* * 停止 *
+
停止 CIM 服务器。

* * 重新启动 *
+
重新启动 CIM 服务器。

* * 状态 *
+
获取 CIM 服务器的状态。
