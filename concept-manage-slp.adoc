= 管理 SLp
:icons: font
:imagesdir: ./media/


[role="lead"]
此 slp 服务广播 WBEM 服务。启用此 slp 服务后，客户端应用程序可以发现此 CIMOM 服务器。您也可以使用 `slp.conf` 文件指定 slp 配置设置。

如果尚未启用此 slp 服务，则可以使用 `smis slpd start` 命令启动此 slp 服务。要停止此 slp 服务，请使用 `smis slpd stop` 命令。
