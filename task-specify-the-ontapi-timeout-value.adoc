= 指定 ONTAPI 超时值
:icons: font
:imagesdir: ./media/


[role="lead"]
SMI-S 提供程序对存储系统进行 ONTAP API （ ONTAPI ）调用。默认情况下， ONTAPI 超时为 300 秒。您可以将超时设置为介于 60 到 300 秒之间的值。



== 开始之前

* 您必须已具有管理员登录凭据。
* 您必须已以管理员身份登录到主机系统。




== 步骤

. 使用新的超时值（以秒为单位）创建名为 `ONTAPI_timeout_SEC` 的系统或用户环境变量。
+
有关创建环境变量的信息，请参见 Windows 文档。
