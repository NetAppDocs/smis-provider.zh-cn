= SMIS 磁盘
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
``smis disks`` 命令可显示存储系统的磁盘信息。`smis disks` 仅在与 Data ONTAP 7- 模式控制器结合使用时才起作用。

SMIS 磁盘

` [-t ｛ * http* | * https * ｝ ]`

`C ： \Program Files （ x86 ） \NetApp\SMIs\pegasus\bin`

具有有效用户名和密码的用户

.Parameters
* ` * （ -t ｛ http ： https ｝ ） *`
+
要使用的协议： HTTPS （默认）或 HTTP



`smis disks` 命令及其缩写输出：

[listing]
----
smis disks
