= SMIS 列表
:icons: font
:imagesdir: ./media/


[role="lead"]
`smis list` 命令可显示已添加的存储系统。



== 语法

SMIS 列表

` [-t ｛ * http* | * https * ｝ ]`



== 位置

`C ： \Program Files （ x86 ） \NetApp\SMIs\pegasus\bin`



== 权限级别

具有有效用户名和密码的用户



== Parameters

* ` * （ -t ｛ http ： https ｝ ） *`
+
要使用的协议： HTTPS （默认）或 HTTP





== 示例

`smis list` 命令及其输出：

[listing]
----
smis list
ONTAP_FilerData.hostName="10.16.180.122",port=80
----
