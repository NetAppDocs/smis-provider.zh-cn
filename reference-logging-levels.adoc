= 日志记录级别
:icons: font
:imagesdir: ./media/


[role="lead"]
您可以指定所记录的消息类型（例如，您只希望记录致命的系统消息）。

您可以将日志记录级别配置为以下级别之一：

* * 跟踪 *
+
将跟踪消息保存在 cimserver_standard 日志中。

* * 信息 *
+
记录所有（信息性，警告，严重和致命）系统消息。

* * 警告 *
+
记录警告，严重和致命系统消息。

* * 严重 *
+
记录严重和致命系统消息

* * 致命 *
+
仅记录致命系统消息。
