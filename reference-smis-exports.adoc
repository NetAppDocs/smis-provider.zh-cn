= SMIS 导出
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
`smis exports` 命令可显示存储系统的网络连接存储（ NAS ）导出。

SMIS 导出 ` [-t ｛ * http* | * https * ｝ ]`

`C ： \Program Files （ x86 ） \NetApp\SMIs\pegasus\bin`

具有有效用户名和密码的用户

.Parameters
* ` * （ -t ｛ http ： https ｝ ） *`
+
要使用的协议： HTTPS （默认）或 HTTP
