= slptool findsrvs
:icons: font
:imagesdir: ./media/


[role="lead"]
`slptool findsrvs` 命令可查找在网络上运行的 WBEM 服务。

`slptool findsrvs _service_`

`C ： \Program Files （ x86 ） \NetApp\SMIs\pegasus\bin`

具有有效用户名和密码的用户

.Parameters
* ` * _service_*`
+
指定服务类型。
