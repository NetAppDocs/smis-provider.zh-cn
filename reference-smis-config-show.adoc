= SMIS config show
:icons: font
:imagesdir: ./media/


[role="lead"]
`smis config show` 命令可列出当前的 CIM 服务器配置信息。

`smis config show`

`C ： \Program Files （ x86 ） \NetApp\SMIs\pegasus\bin`
