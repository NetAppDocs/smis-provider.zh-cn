    SMIS CRP.

    05/03/2022 提供者

    smis 命令可描述 NetApp SMI-S Provider 支持的已注册 CIM 配置文件，包括 NetApp SMI-S Provider 配置文件。

    SMIS CRP.

    ` [-t ｛ * http* | * https * ｝ ]`

    C ： \Program Files （ x86 ） \NetApp\SMIs\pegasus\bin

    具有有效用户名和密码的用户

    Parameters

    • ` * （ -t ｛ http ： https ｝ ） *`

      要使用的协议： HTTPS （默认）或 HTTP
